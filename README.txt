Antes de inicializar o projeto é necessário a instalação do .NET CLI (https://docs.microsoft.com/en-us/dotnet/core/tools/) e do Node.js (https://nodejs.org/en/).

Para inicializar a aplicação front-end é necessário instalar as dependências do projeto com o comando 'npm i' na pasta do projeto 'frontend'. Após a instalação das dependências é só utilizar o comando 'npm start'.

Para inicializar a aplicação back-end é necessário utilizar o comando 'dotnet run' na pasta do projeto 'api'.
